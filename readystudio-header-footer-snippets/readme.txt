Add scripts/styles to header/body/footer and manage PHP/CSS/JS/HTML snippets safely. Includes Safe Mode, import/export, CPT-based snippet manager, and AJAX toggle in list table.

== Description ==
- Global fields for `<head>`, body-open and footer.
- Snippet Manager (CPT): types PHP/CSS/JS/HTML, locations header/body/footer/manual, Active & Priority.
- Safe Mode (persistent & 15-min temporary).
- Import/Export JSON.
- Shortcode: `[rshf_snippet id="123"]`.
- Transient caching for performance.
- Brand: Ready Studio (#00b0a4).

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin.
3. Go to **Ready HF** menu.
